## 🚀 Featured Project

### Calourie AI

AI-powered nutrition tracking application built with Kotlin and Jetpack Compose, designed around Clean Architecture and an offline-first experience.

**✨ Highlights**
- Barcode scanning using ML Kit and CameraX
- Nutrition lookup through OpenFoodFacts API
- Manual meal logging and calorie tracking
- BMR and TDEE-based goal calculation
- Local persistence with Room Database
- Modern Material 3 user interface

**🧩 Tech Stack**
Kotlin • Jetpack Compose • Clean Architecture • MVVM • Hilt • Room • Retrofit • ML Kit • CameraX • Coroutines • Flow

**📦 Repository**
https://github.com/saqibcheema/calourie_ai
